Open Discussion su Sports Open Data API Restfulness

Inviato da Sports Open Data il Gio, 04/14/2016 - 22:15
Tipo di attività: 
Civic Hackathon(s) 07/05/2016
Livello di difficoltà: 
Intermedio
Durata: 
60 minuti

Sports Open Data è una giovane associazione culturale senza scopo di lucro che si pone l'obiettivo di fornire dati statistici in modalità open data attraverso delle API Rest.
Questo hackaton vuole essere una open discussion, quindi osservazioni aperte su:
- metodi esistenti con proposta di modifiche sia alle richieste che alle response dei metodi
- proposta di nuovi metodi
- restfullness dei metodi proposti
Durante l'hackaton sarà possibile chiamare ogni singolo metodo API semplicemente tramite un client API Rest (es: postman).
Tutte le osservazioni saranno conservate per migliorare il servizio API dell'associazione al fine di elevare ancora di più la qualità dei dati proposti.
