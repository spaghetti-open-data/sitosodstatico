Sappi-a-Mo portale di giornalismo Open Data under construction

Inviato da Laura Camellini il Mar, 02/03/2015 - 11:54
Tipo di attività: 
(SOD15) Barcamp 27/03/2015
Livello di difficoltà: 
Intermedio
Durata: 
30 minuti

Il nostro Portale di Open data  
http://www.jeeltcraft.com/opendata-modena/
E' ancora in costruzione, ma gli strumenti che abbiamo studiato sono innnovativi e possono aiutare molti a creare siti web ed app con i dati aperti.
guardate la DEMO troverete i link a tutti i repository specifici su github, per aiutarci a create un set di strumenti per spaghetti open data, l'elemento da installare con bower per poter usare tutti i charts e' questo:
https://github.com/jilt/sheets-charts
Abbiamo creato un repository specifico per ognuno degli elementi che abbiamo sviluppato, perche' tuta la comunita' di sod possa sfruttare le potenzialita delle google apis ed intrecciarle con i web components.
Potremo costruire le nostre visualizzazioni direttamente con i fogli di google, usando pattern di crawling per il web come (xpath) e le api che la lista SOD ha sviluppato nell'ultimo anno per interrogare i database degli open data.
Se arvete domande su come usare i web component che abbiamo sviluppato fatevi avanti, abbiamo seguito le direttive del material design per lo sviluppo grafico del portale.
A breve sara' disponibile anche la pagina su github.
A venerdi!
PS: per adesso abbiamo rinunciato a sviluppare su d3.js ma il nostro elemente mappa (bower install --save map-sheet) puo' essere manipolato facilmente con d3.js ed e' ancora ad uno stadio embrionale, potremmo lavorare proprio su quello:
https://www.youtube.com/watch?v=wqPGFs0cqxI
http://bl.ocks.org/mbostock/899711
In alternativa si puo' definire meglio il web component per i geochart a marker di google
https://developers.google.com/chart/interactive/docs/gallery/geochart#Ma...
che ancora non funziona e non e' stato inserito nella demo, questo e' il repo github: 
https://github.com/jilt/sheets-marker-chart
Ci vediamo a SOD!
